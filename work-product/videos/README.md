# Video Assets

Each week has a stable folder for raw recordings, exported videos, thumbnails,
transcripts, and platform links.

Recommended naming:

```text
videos/week-01/
|-- README.md
|-- links.md
|-- transcript.md
|-- thumbnail.png
`-- exports/
    |-- bilibili.mp4
    `-- youtube.mp4
```

Do not commit very large video exports if the repository policy or hosting
limits make that impractical. In that case, commit `links.md` and keep the
external video URL in the matching book chapter.
