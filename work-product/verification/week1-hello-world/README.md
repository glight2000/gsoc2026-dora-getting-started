# Week 1 Hello World Verification

This example verifies the Week 1 dora-rs installation and Hello World dataflow.

Run from this directory:

```powershell
./run.ps1
```

The script creates `.venv`, installs `requirements.txt`, runs the dataflow for
four seconds, and checks for `listener received: Hello from dora-rs`.

Generated files:

- `.venv/` is local and ignored by git.
- `logs/*.log` is local and ignored by git.
- `out/` is generated by Dora and ignored by git.
- Sanitized `logs/*.md` notes can be committed.
